Piracy

Piracy is essentially a web hosted file server with a heap of illegally distributed files that anyone can collect and download via torrenting programs (eg. BitTorrent, uTorrent) using p2p (peer-to-peer) transferring.

Legal, Social and Ethical Issues

Piracy is illegal, but the laws behind it are very technical and very vague. If you follow the Australian Government laws, it’s illegal to both download and distribute the files, as long as you’re connected to $ISP. If you follow the American Government laws, it’s only illegal to distribute the files, not download them.

Digital Systems

A common way to pirate music, movies, software, games etc. is through torrenting. Applications such as uTorrent and BitTorrent are used to obtain these files through peer-to-peer transfer, which relies on users to seed the segments of the file being downloaded to others. Ways to do this privately as according to the government it's illegal, therefore people paranoid about being 'caught' use a VPN (Virtual Private Network) to essentially mask their IPv6 (public) address.

Key Stakeholders and their Responsibilities

The key stakeholders for the issue of piracy would be those who pirate, and those who don’t pirate. Their responsibilities are to not distribute the downloaded files.

Positive and Negative Opinions of Stakeholders

Opinions of a pirate would probably be along the lines of “I get stuff for free, you don’t.”
Opinions of someone who doesn’t pirate would be a “I support the $billionDollarFranchise”

Storing and Communicating Data

Some options for storing the data could be site-hosting companies, like github (which is what you’re reading this on right now.) Some advantages could be that it’s easy for everyone to contribute to the website development, everyone can see what data is being added in real time, and everyone can make changes overall. Some disadvantages could be that it’s solely on the internet, meaning it’s not really able to be changed and edited from somewhere that doesn’t have internet, and it’s hard to fix if there is problems, especially if it’s github and we need to contact an admin to fix it for us.
